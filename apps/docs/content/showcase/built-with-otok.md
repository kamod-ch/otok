# Built with Otok

The playground is the current in-repository reference application. It demonstrates:

- SSR
- file routes
- loaders
- actions
- progressive forms
- route middleware
- typed routes
- islands
- soft navigation
- Node production build

Planned reference applications:

- Kamod AI Audit
- Kamod Flat CMS

These projects should validate Otok outside the playground before being presented as production showcases.
